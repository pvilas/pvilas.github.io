--- layout: post title: Django Guardar im√°genes desde string date: 2011-02-24 17:52:40 UTC comments: false categories: django --- Para guardar una imagen en un campo de tipo ImageField de Model desde una cadena como un POST http o desde memoria hacemos lo siguiente:


from django.core.files.uploadedfile import SimpleUploadedFile
import cStringIO

buf = cStringIO.StringIO()
# adquirimos imagen y la guardamos en buf i.e. mediante pycurl

foto = SimpleUploadedFile('imagen.jpg', buf.getvalue(), "image/jpeg")

# ahora foto contiene el stream pero en formato 'fichero'

p=Modelo( imagen=foto )
p.save()
